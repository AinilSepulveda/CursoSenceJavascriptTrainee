
# Estructura del Proyecto

Este proyecto está organizado de manera simple y clara para facilitar la gestión de archivos y mejorar la mantenibilidad del código.

## 📁 Estructura de Carpetas

```
.
├── index.html              # Página principal del proyecto
├── readme.md               # Este archivo (documentación)
└── assets/                 # Carpeta que contiene todos los recursos
    ├── css/
    │   └── style.css       # Estilos CSS del proyecto
    ├── js/
    │   └── main.js         # Scripts JavaScript del proyecto
    └── img/                # Imágenes utilizadas en el proyecto
```

## 📄 Archivos Principales

### `index.html`
Página principal del proyecto que contiene:
- **Header**: Título principal "Orden de estructura de proyecto"
- **Section**: Sección con artículo que explica la estructura de carpetas
- **Aside**: Barra lateral con enlaces de interés
- **Footer**: Pie de página con información del autor

### `assets/` - Recursos del Proyecto

#### `assets/css/`
Contiene los archivos de estilos CSS para dar formato visual al proyecto.
- **style.css**: Hoja de estilos principal

#### `assets/js/`
Contiene los archivos JavaScript para interactividad.
- **main.js**: Script principal que modifica dinámicamente el contenido

#### `assets/img/`
Carpeta para almacenar todas las imágenes utilizadas en el proyecto.

## 🚀 Cómo Usar

1. Abre el archivo `index.html` en tu navegador
2. El contenido se cargará automáticamente con los estilos de `style.css`
3. Los scripts de `main.js` se ejecutarán automáticamente (con el atributo `defer`)

## 📝 Notas

Esta organización de carpetas permite:
- ✅ Mantener los archivos ordenados por tipo de contenido
- ✅ Facilitar la búsqueda de archivos específicos
- ✅ Mejorar la mantenibilidad del proyecto a largo plazo
- ✅ Escalar el proyecto de manera ordenada
