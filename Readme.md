📚 Ejercicios Curso HTML
📌 Descripción

Este proyecto contiene un conjunto de ejercicios desarrollados durante el curso de HTML.
El archivo index.html funciona como un menú principal para acceder a cada actividad.

📂 Estructura

Cada botón dirige a una carpeta con su respectivo ejercicio:

Actividad M2 L1

Actividad M2 L2

Actividad M2 L6

Lección 2 (ejercicios 2 y 3)

Lección 4

Lección 5

Evaluación final

🛠️ Tecnologías

HTML5

CSS3

Bootstrap 5

🚀 Uso

Abrir index.html en un navegador

Seleccionar cualquier botón para acceder a los ejercicios

📎 Notas

El proyecto es solo frontend

Cada carpeta contiene ejercicios independientes
